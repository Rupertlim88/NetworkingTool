<blockers>
- **GSAP stacking context constraint**: Any effect that needs to render BEHIND a card cannot use pseudo-elements (::before/::after) because GSAP's translateZ() creates stacking contexts. Must use box-shadow or sibling DOM elements instead. This is a permanent constraint for this project.
- **Dark-on-dark contrast**: Shadows on teal/midnight backgrounds need 0.8-0.95 opacity to be visible. Standard 0.3-0.4 values are invisible. Always test visibility on dark backgrounds.
</blockers>
