**4. Configure Tailwind CSS theme tokens (LOCKED DECISION):**
Both themes must be defined as CSS custom properties so switching themes is a single class change on `<html>`. Configure `tailwind.config.ts` to extend the theme with the following CSS variable-based tokens:

In `tailwind.config.ts`, extend the theme with custom colors:
```typescript
import type { Config } from 'tailwindcss'

const config: Config = {
  content: [
    './pages/**/*.{js,ts,jsx,tsx,mdx}',
    './components/**/*.{js,ts,jsx,tsx,mdx}',
    './app/**/*.{js,ts,jsx,tsx,mdx}',
  ],
  theme: {
    extend: {
      colors: {
        'grid-bg': 'var(--color-grid-bg)',
        'card-surface': 'var(--color-card-surface)',
        'accent-primary': 'var(--color-accent-primary)',
        'accent-secondary': 'var(--color-accent-secondary)',
      },
    },
  },
  plugins: [],
}
export default config
```

In `app/globals.css`, add the theme definitions under `@layer base`:
```css
@layer base {
  /* Theme C — Rich Teal + Soft White (default) */
  :root,
  [data-theme="teal"] {
    --color-grid-bg: #071a2e;
    --color-card-surface: rgba(255, 255, 255, 0.08);
    --color-accent-primary: #f59e0b;
    --color-accent-secondary: #22d3ee;
  }

  /* Theme A — Midnight Blue + Warm Coral */
  [data-theme="midnight"] {
    --color-grid-bg: #0a0f1e;
    --color-card-surface: #111827;
    --color-accent-primary: #ff6b6b;
    --color-accent-secondary: #4fc3f7;
  }
}
```

Theme switching is done by setting `data-theme="teal"` or `data-theme="midnight"` on the `<html>` element. The animation overlay always uses pure black (`#000000`) and is NOT theme-dependent — do not add a theme variable for the animation background.

IMPORTANT: Do NOT install framer-motion. Do NOT use src/ directory. Do NOT create a subdirectory.
