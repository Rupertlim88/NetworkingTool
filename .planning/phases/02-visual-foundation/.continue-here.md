## Phase 02 Context Discussion — Areas Completed

### 1. Contact Card Design (fully discussed)
- Portrait rectangle 3:4, rounded-2xl, circular photo on card surface
- Name below photo, stars centered at bottom strip
- Layered shadow (2-3 layers) + hover glow (accent color) + scale(1.02) on hover
- Z-axis pop entrance animation with stagger (GSAP ScrollTrigger)
- 30 placeholder contacts using user's own headshot photo

### 2. Popup Overlay (fully discussed)
- Card-expand animation (~300-400ms ease-out) to open
- Near-black #0d0d0d surface, accent gradient glow at top (theme-dependent)
- Max-width 550px, max-height 80vh, responsive
- Three close methods: click outside, X button (top-right), Escape
- Collapsible bio/resume section (summary + first experience default, expand for full)
- Level Up button: own row, full-width, solid accent fill — primary CTA
- Action icons row: LinkedIn (new tab), Gmail (compose with email pre-filled), More Info (popup expands to full page -> /contacts/[id])
- Auto-hiding scrollbar (accent colored) + bottom gradient hint
- Full color spec for both themes locked (tags, buttons, icons, glow)

### 3. Grid Layout (fully discussed)
- 5 columns desktop, 3 tablet, 2 mobile
- ~1400px max-width container, 20px gaps, ~254x339px cards
- Page scrolls naturally, no visible scrollbar on grid
- Grid is home page (/)
- Auto-hiding header bar: logo left, [All | Blue Stars] filter toggle center, gear icon right
- Default sort: by star level (3->2->1), Blue filter shows only blue-star contacts

### 4. Grid Background (fully discussed)
- Base theme color stays (teal #071a2e or midnight #0a0f1e)
- Accent-color gradient bleed at top: theme accent colors at 10-15% opacity, fading to pure base in middle-bottom
- No contrast issues at 10-15% opacity — cards sit in calmer area

### 5. Theme Switcher (partially discussed)
- Lives on /settings page (separate route)
- Neutral dark background (#0d0d0d)
- Two visual swatch cards with vertical divider, filled with theme colors
- Labels: "Teal" and "Midnight"
- Active theme highlighted (border or checkmark)
- Instant theme switch (no animation)
- localStorage persistence (Phase 1 decision)
